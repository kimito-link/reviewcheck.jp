# Chrome Web Store 掲載情報（口コミチェック.jp）

## 拡張機能名
口コミチェック.jp｜Google口コミチェッカー

## 概要（短い説明・132文字以内）
Googleマップで開いている店舗のGoogle口コミ（星評価・口コミ数・競合との差・あと何件で追いつけるか）をワンクリックで簡易チェック。口コミ改善・MEO対策のヒントが分かります。

## 詳細な説明
「このお店、口コミは足りている？」——Googleマップで店舗ページを見ているときに、ツールバーのアイコンを押すだけで、その店舗のGoogle口コミの状態を簡易診断できる拡張機能です。

【できること】
・現在の星評価・口コミ数を表示
・0〜100点の「選ばれやすさスコア」を表示
・目標評価に近づくために「あと何件の高評価口コミが必要か」の目安を表示
・詳しい診断結果（競合比較・改善ポイント）をWeb版で確認
・口コミ改善・MEO対策の相談導線

【こんな方に】
・店舗オーナー、クリニック・飲食店・美容院・整体院・士業の方
・自店舗や競合店舗の口コミ状況をサッと把握したい方
・Google口コミ・MEOの改善を検討している方

【プライバシー・権限について】
・権限は activeTab のみ。閲覧履歴を常時収集することはありません。
・診断は「この店舗を診断」ボタンを押したときだけ、現在のタブのURLを当サービスのAPIに送信して行います。
・口コミの購入・偽レビュー・やらせ・競合への低評価工作などの不正行為は一切行いません。Googleのポリシーに沿った正当な口コミ改善を前提としています。

【ご注意】
本診断は公開情報・入力情報をもとにした簡易診断です。検索順位・来店数・口コミ増加・星評価の改善を保証するものではありません。表示される数値は目安です。
