# プライバシーポリシー（草案）

> 実運用ページは `apps/web/app/privacy/page.tsx`（`/privacy/`）。本ファイルはレビュー・差し替え用の元原稿。

口コミチェック.jp（以下「当サービス」）は、利用者のプライバシーを尊重し、以下の方針で個人情報・入力情報を取り扱います。

## 1. 取得する情報
当サービスは、診断のために利用者が入力した店舗名・GoogleマップURL・星評価・口コミ数・競合情報等を取得します。閲覧履歴を常時収集することはありません。

## 2. 利用目的
取得した情報は、当サービスの簡易診断および結果表示のためにのみ使用します。診断結果の共有機能を利用する場合に限り、入力内容を含む共有用URLを生成します。

## 3. 外部サービスへの照会
店舗情報の取得のため、Google Places API 等の外部サービスへ店舗名・GoogleマップURL・Place ID 等を照会する場合があります（接続している場合）。当サービスは、口コミの購入・偽レビュー・やらせ・競合への低評価工作などの不正行為は一切行いません。

## 4. 第三者提供
法令に基づく場合を除き、取得した情報を第三者に提供しません。

## 5. Cookie・アクセス解析
サービス改善のためアクセス解析を利用する場合があります。個人を特定する目的では使用しません。

## 6. お問い合わせ
本ポリシーに関するお問い合わせは、support@reviewcheck.jp または当サイトの相談・お問い合わせ窓口までご連絡ください。

## 7. 改定
本ポリシーは予告なく改定されることがあります。改定後は本ページに掲載した時点で効力を生じます。

---
※本ポリシーは草案です。運営主体の正式名称・所在地・連絡先・実際のアクセス解析ツール等に合わせて加筆してください。
