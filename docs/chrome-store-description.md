# Chrome Web Store 提出用 説明文

> 拡張本体の掲載原稿は `apps/chrome-extension/STORE-LISTING.md` と同期。提出時はこちらをコピーして使用。

## 拡張機能名
口コミチェック.jp｜Google口コミチェッカー

## 概要（132文字以内）
Googleマップで開いている店舗のGoogle口コミ（星評価・口コミ数・競合との差・あと何件で追いつけるか）をワンクリックで簡易チェック。口コミ改善・MEO対策のヒントが分かります。

## 詳細説明
「このお店、口コミは足りている？」——Googleマップで店舗ページを見ているときに、ツールバーのアイコンを押すだけで、その店舗のGoogle口コミの状態を簡易診断できます。

できること
・現在の星評価・口コミ数を表示
・0〜100点の「選ばれやすさスコア」を表示
・目標評価に近づくために「あと何件の高評価口コミが必要か」の目安を表示
・詳しい診断結果（競合比較・改善ポイント）をWeb版で確認
・口コミ改善・MEO対策の相談導線

こんな方に
・店舗オーナー、クリニック・飲食店・美容院・整体院・士業の方
・自店舗や競合店舗の口コミ状況をサッと把握したい方

権限・プライバシー
・権限は activeTab のみ。閲覧履歴を常時収集しません。
・「この店舗を診断」ボタンを押したときだけ、現在のタブのURLを当サービスのAPIに送信します。
・口コミ購入・偽レビュー・やらせ・競合への低評価工作などの不正行為は一切行いません。

ご注意
本診断は簡易診断であり、検索順位・来店数・口コミ増加・星評価の改善を保証しません。数値は目安です。

## カテゴリ / 言語
仕事効率化 / 日本語

## 単一用途の説明（Chrome Web Store 審査用）
本拡張は「現在開いているGoogleマップの店舗ページのGoogle口コミ状態を簡易診断する」という単一目的のために、ユーザー操作時にアクティブタブのURLを取得し、自社診断APIへ送信して結果を表示します。

## データ使用の開示
- 取得: 現在のタブのURL（ユーザーが診断ボタンを押したときのみ）
- 送信先: 自社API（reviewcheck.jp）
- 用途: 当該店舗の口コミ簡易診断の実行と結果表示
- 販売しない／無関係な第三者へ譲渡しない／信用調査等に使用しない
